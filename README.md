# Proxies.Rent Dashboard

Мультитенантная платформа управления мобильными прокси с биллингом и интеграцией Точка Банка.

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Backend | Node.js (≥18), Express.js |
| База данных | SQLite (better-sqlite3, WAL mode) |
| Аутентификация | bcrypt, токен-сессии |
| Безопасность | helmet, express-rate-limit |
| Банк | Tochka Bank API (JWT, webhook) |

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Продакшн
npm start

# Разработка (auto-reload)
npm run dev

# Миграция из JSON (при первом переходе на SQLite)
node migrate.js
```

### Принцип работы

- При старте загружает данные из SQLite в память
- Мутации пишутся сразу в SQLite (prepared statements) и обновляют in-memory cache
- Внешние API прокси-серверов опрашиваются периодически и кешируются
- Банковские платежи приходят через webhook

## Управление клиентами

### Профили
- Уникальный логин, хеширование пароля (bcrypt)
- Привязка к порту прокси-сервера (`portName`)
- Контактная информация, заметки
- API-ключ для программного доступа
- Реферальная система (код, баланс)

### Модели биллинга

| Тип | Описание |
|-----|----------|
| `per_gb` | Оплата за потреблённый трафик (руб/ГБ) |
| `per_modem` | Фиксированная ежемесячная плата за модем |
| `flat` | Фиксированная ежемесячная подписка |

### Реквизиты
- ИНН, КПП, юридическое название, адрес
- Информация о договоре
- Авто-генерация актов и счетов (переключатели per client)
- Тип клиента: юр. лицо / физ. лицо

### Баланс и валюта
- Текущий баланс отслеживается в реальном времени
- Поддержка нескольких валют (по умолчанию RUB)
- Ежедневные автоматические списания по тарифу

## Управление прокси и модемами

### Прокси
- HTTP и SOCKS5 протоколы
- Мульти-серверная архитектура (неограниченное количество серверов)
- Динамическая аллокация портов
- Генерация учётных данных (login/password)
- Ротация IP с историей

### Модемы
- Статус: online / rotating / offline / rebooting
- IMEI, тип сети (LTE/3G/2G), уровень сигнала
- USB-reset, перезагрузка, SMS/USSD
- Speedtest с историей результатов
- Отслеживание uptime

## Аналитика трафика

### Потребление
- Ежедневный трафик (bytes in/out) по каждому порту
- Месячная агрегация по клиенту
- Графики потребления с навигацией по месяцам
- Кастомный диапазон дат

### Ресурсы
- Top hosts/домены по частоте обращений (глобально и per client)
- Bandwidth reports (однодневные, периодные)
- Отчёты по странам

### Периоды аналитики (админ)
- Сегодня, Вчера, Месяц, Предыдущий месяц, Всё время

## Биллинг и платежи

### Ledger (журнал операций)
- Полная история всех начислений и оплат
- Типы записей: `charge` (списание), `payment` (оплата), `correction` (корректировка)
- Баланс до/после каждой операции
- Связь с GB трафика и количеством модемов

### Платежи
- Ручное добавление
- Автоматический зачёт из банковских платежей (по ИНН)
- История с сортировкой по дате (новые сверху)
- Источник платежа: manual, bank, API

### Документы
- **Акты** (closing documents) — автогенерация на конец месяца
- **Счета** (bills) — автогенерация для выставления
- Загрузка/скачивание произвольных документов
- Подписание через Точка Банк API

## Интеграция Точка Банк

### Аутентификация
- JWT-токен для API-запросов
- JWKS-кеширование для верификации подписей
- RSA-SHA256 проверка webhook

### Функционал
- Webhook приёма входящих платежей
- Автоматический мэтчинг по ИНН плательщика → клиент
- Авто-зачисление на баланс (опционально)
- Генерация и отправка актов через API
- Генерация и отправка счетов через API
- Скачивание PDF документов
- Отклонение нераспознанных платежей

### Документооборот
- Формирование актов с детализацией услуг
- Русские названия месяцев для документов
- Поддержка ИП и юридических лиц
- Номерация актов и счетов

## Безопасность

- **Аутентификация**: сессионные токены с TTL, заголовок `x-auth-token`
- **Хеширование**: bcrypt для паролей
- **Rate limiting**: настраиваемый лимит попыток входа, ограничение API-запросов
- **Helmet**: HTTP security headers
- **Мультитенантность**: фильтрация по `portName` — каждый клиент видит только свои данные
- **Аудит**: полный лог действий администратора (хранение 90 дней)
- **Импersonation**: администратор может войти от лица клиента (логируется)
- **Token reset**: механизм сброса пароля через токен

## База данных

SQLite с 16 таблицами, WAL mode, foreign keys:

| Таблица | Назначение |
|---------|-----------|
| `clients` | Профили клиентов, биллинг, реквизиты |
| `payments` | Оплаты клиентов |
| `billing_ledger` | Журнал всех операций |
| `sessions` | Активные сессии |
| `bank_payments` | Входящие банковские платежи |
| `client_documents` | Загруженные документы |
| `closing_documents` | Акты (closing docs) |
| `bills` | Счета на оплату |
| `audit_log` | Аудит действий (90 дней TTL) |
| `ip_tracking` | Текущий IP каждого прокси |
| `ip_history` | История изменений IP |
| `uptime_tracking` | Uptime модемов |
| `daily_traffic` | Трафик по дням (port + date) |

### Автомиграция
При первом запуске с пустой БД и существующими JSON-файлами автоматически запускается `migrate.js`.

## Админ-панель (admin.html)

Одностраничное приложение с вкладками:

1. **Модемы** — статусы, управление (reset, reboot, SMS/USSD), speedtest
2. **Аналитика** — трафик по регионам/клиентам/ресурсам, матрица, периоды (сегодня/вчера/месяц/пред.мес/всё время)
3. **Клиенты** — список с балансами, карточка (создание/редактирование), оплаты, документы
4. **Документооборот и операции** — акты, счета, банковские платежи, мэтчинг
5. **Настройки** — тарифы, расписание, конфигурация Точка Банк

### Возможности
- Тёмная/светлая тема
- Настраиваемые колонки (сохранение в localStorage)
- Поиск клиентов с пагинацией
- Массовые операции (ежедневное списание, генерация актов/счетов)
- Аудит-лог всех действий

## Личный кабинет (index.html)

Клиентский портал с вкладками:

1. **Трафик** — таблица прокси с реквизитами, ротация IP
2. **Аналитика** — график потребления по дням, навигация по месяцам
3. **Баланс** — история биллинга, текущий баланс
4. **Документы** — скачивание загруженных документов
5. **API** — документация v1 API, ключ доступа
6. **Реферальная программа** — код, баланс, условия
7. **Инструменты** — проверка прокси, конвертер форматов
